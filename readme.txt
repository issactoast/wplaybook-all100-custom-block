A simple plugin to add a beautiful shimmer effect to your buttons using the 'wplaybook-button' class.

== Description ==

This plugin enqueues a lightweight CSS file that adds a shimmering animation to any element with the `wplaybook-button` class. 

Simply add the class `wplaybook-button` to your buttons, links, or button blocks in the WordPress editor, and watch them shine!

== Installation ==

1. Upload the `wplaybook-shimmer-button` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the class `wplaybook-button` to any element you want to apply the shimmer effect to (e.g. in Gutenberg, add it to 'Additional CSS class(es)' under Advanced settings).

== Frequently Asked Questions ==

= How do I use the shimmer button? =

Just add the class `wplaybook-button` to any block or HTML element in WordPress.
